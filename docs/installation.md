# Installation

## Stable release

To install care_odoo_be, run this command in your terminal:

```sh
pip install care_odoo_be
```

## From source

The source files for care_odoo_be can be downloaded from the [Github repo](https://github.com/ohcnetwork/care_odoo_be).

You can either clone the public repository:

```sh
git clone git://github.com/ohcnetwork/care_odoo_be
```

Or download the [tarball](https://github.com/ohcnetwork/care_odoo_be/tarball/master):

```sh
curl -OJL https://github.com/ohcnetwork/care_odoo_be/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_odoo_be
pip install .
```
